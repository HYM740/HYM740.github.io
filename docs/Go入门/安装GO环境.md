# 安装 Go 环境

做任何事都需要一些前置条件。比如想要学计算机，首先的条件就是得有一台计算机。

想要学习 Go 语言，则必须先在计算机上安装 Go。

Go语言支持以下操作系统

- MircoSoft Windows
    - Windows 7 及后续版本
- Apple macOS
    - macOS 10.13 及后续版本
- Linux
    - 内核版本 2.6.32 及后续版本

当然如果有能力且有必要的情况下，也可以使用官方提供的源代码进行构建，从而在其他平台上使用 Go 语言。
